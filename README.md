# TodoManager

**TodoManager** 是一款基于 Android 现代开发栈开发的个人待办事项管理器。它使用 **Kotlin** 语言与声明式 UI 框架 **Jetpack Compose** 构建，旨在提供流畅、直观的任务记录与管理体验，不同于传统的日历和闹钟应用。

## 📱 页面功能详解

### 主页

事项管理的核心界面，展示所有待办事项。

- **多级任务展示**：以层级结构展示主任务及其子任务。
- **快捷状态切换**：点击复选框即可快速标记任务完成或未完成。
- **日历跳转**：点击日期即可展开日历，点击不同日期以跳转。显示其他日期的未完成任务。
- **完成动画**：当用户完成当日所有任务时，附赠彩炮Boom。

### 事项编辑页

创建新事项 或 编辑现有事项。

- **详情配置**：修改任务标题、内容等基本信息。
- **提醒开关**：设置或关闭任务提醒。
- **子事项管理**：为此事项添加或删除子任务。
- **闹钟设置**：提供类似日历的定点提醒功能，确保准时通知。



## 📁 项目结构

```
com.sheepspacefly.todomanager
├── MainActivity.kt                    # 应用主入口，承载导航和主界面
├── ui
│   ├── theme
│   │   ├── Color.kt                   # 自定义颜色资源 (如 AmberYellow, DarkText)
│   │   ├── Effect.kt                  # 自定义视觉效果（如彩炮动画）
│   │   ├── Theme.kt                   # Material3 主题与全局水波纹效果配置
│   │   └── Type.kt                    # 字体排版样式定义
│   └── ...
├── AlarmManager.kt                    # 闹钟广播接收器，处理定时唤醒
├── NotificationHelper.kt              # 通知构建与管理工具类
└── TodoViewModel.kt                   # 核心 ViewModel，负责：
                                        #   - 数据持久化（Room）
                                        #   - 任务的增删改查逻辑
                                        #   - 闹钟事件的统一管理
```



## 🚀 运行说明

提供可直接安装的 APK 文件。为确保 **后台通知提醒** 正常运作，安装后请开启以下权限：

- **自启动权限**：允许应用在后台运行，确保即使应用被清理，闹钟提醒依然能准时触发通知。
- **通知管理权限**：请务必开启 **悬浮通知权限** 与 **桌面图标角标权限**，以提供类似QQ、微信的通知效果。

***特别注意***：如果开启了上述权限仍然无法正常通知，请确保开启类似下图 **红圈** 中的设置项内的权限，此设置项在触发了第一次通知后会出现。
